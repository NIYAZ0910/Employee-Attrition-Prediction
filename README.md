# Employee Attrition Analysis

## 📊 Project Overview
This project analyzes employee attrition using an HR dataset to understand the factors that influence employee turnover. The goal is to identify patterns in workforce data and provide insights that can help organizations improve employee retention.

## 🎯 Objectives
- Analyze employee attrition trends
- Identify factors affecting employee turnover
- Explore relationships between job role, income, satisfaction, and attrition
- Provide data-driven insights for HR decision making

## 📁 Dataset
The dataset contains employee information such as:
- Age
- Department
- Job Role
- Monthly Income
- Years at Company
- Job Satisfaction
- Work Environment
- Attrition Status

## 🛠 Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🔍 Analysis Steps
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insights and Findings

## 📈 Key Insights
- Certain departments experience higher attrition rates.
- Job satisfaction and income levels impact employee retention.
- Employees with fewer years at the company tend to leave more frequently.

## ✅ Conclusion
The analysis highlights key factors contributing to employee attrition and provides insights that can help organizations improve retention strategies and workforce management.
